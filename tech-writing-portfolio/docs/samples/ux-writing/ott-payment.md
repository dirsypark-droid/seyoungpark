# OTT 결제 대기 화면 UX Writing 사례

## Context
- 글로벌 OTT 서비스의 결제 대기 화면도 콘텐츠 경험의 일부로 느낄 수 있도록 UX 문구를 개선함

## Before vs. After
![](../../assets/OTT-payment.png)

## Before - UX 이슈
- 기술 중심 표현으로 감정적 여정 단절
- 브랜드 감성 부족  
- 기대감 저하  

## After - UX 개선 사항
- “Almost there!”로 사용자 중심 전환  
- 🍿 이모지로 엔터테인먼트 감성 강화  
- “non-stop fun”으로 결제 후 기대감 유지  
- 글로벌 서비스에 맞는 밝고 친근한 톤앤매너 확립
